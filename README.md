# QR/// Generator

QR Generator statis (tanpa build step, tanpa framework) berbasis browser.

**Generate QR code dari URL atau teks** — semua diproses di perangkat Anda.

## Fitur

- **Dua mode input** — URL atau teks bebas (hingga 1500 karakter)
- **Generate otomatis** — QR muncul langsung saat input diketik, tanpa perlu klik tombol
- **Ukuran fleksibel** — kecil (220px), sedang (320px), besar (420px)
- **Toleransi error (ECC)** — pilih level L / M / Q / H, dilengkapi tabel referensi
- **Warna khusus** — ubah warna kode dan latar via color picker
- **Unduh PNG** — simpan QR sebagai file gambar
- **Salin gambar** — salin QR ke clipboard dengan padding dan warna latar
- **Animasi scan-line** — efek garis pindai khas scanner setiap QR baru
- **100% di sisi klien** — tidak ada data yang dikirim ke server

## Cara pakai

Buka `index.html` langsung di browser, atau gunakan live server bawaan editor/IDE.

## Lisensi

MIT — menggunakan [davidshimjs/qrcodejs](https://github.com/davidshimjs/qrcodejs) sebagai library QR.
